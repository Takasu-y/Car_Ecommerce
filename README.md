# Car Ecommerce
## Description
Vue.jsで構築した簡単な自動車のECサイトです。

## Usage
-categoryのドロップダウン
車種をメーカー毎にフィルターすることが出来ます。

-sort byのドロップダウン
下記3つのオプションで商品をソートすることが出来ます。
* 価格の低い順
* 価格の高い順
* 登録された日付が新しい順
